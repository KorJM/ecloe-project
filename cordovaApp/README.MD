### Home 화면
맨위에는 이미지 슬라이드를 배치하고 시각장애인과 관련된 사진을 넣었음   
그 밑으로 발표했던 목차대로 
아이디어 배경 - Azure - 제작과정, 차별점 - 프로토타입 순으로 배치했음     
그리고 Tab Bar에 홈, 차트, 검색(구글), 새로고침, info 기능을 넣었음
