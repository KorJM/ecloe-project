아이디어 배경


눈을 감고 있을 때 큰 소리나 처음 듣는 소리가 들린다면 불안하고 무섭습니다
이런 고통을 항상 겪고 있는 사람들이 있습니다
바로 시각장애우들 입니다
이런 고통을 겪는 분들은 길거리를 걷는 것 조차 힘든 일입니다.


이런 분들에게 도움을 주는 안내견, 도우미는 굉장히 비싸고 한정적인 자원입니다.
그래서 저희는 시각장애인의 보행에 있어 위험요소를 탐지하고 그것을 방어해줄 수단에 대해 생각해보았습니다.


처음에는 스마트폰을 통해 연속적으로 촬영된 사진을 object detection으로 분석하여 위험요소가 발견되면 사용자가 착용하고 있는 이어폰으로 음성 신호를 주고 특정 버튼 클릭 시 image Captioning과 TTS를 사용해 이미지에 대한 설명을 들려주는 앱을 생각 하였습니다.
그런데 이 방식에는 두 가지의 문제점이 발견되었습니다.

문제점 1


스마트폰을 사용한다면 목에 걸어 놓고 촬영하는 방식인데 그러면 오로지 진행 방향에 대한 정보만 얻을 수 있고 후방의 위험요소에 대한 대처가 힘들어집니다.

문제점 2


시각장애우들은 지팡이로 바닥을 쳤을 때 나는 소리를 들으며 지형지물을 구분하는 등 청각에 의존하는데 이어폰을 끼고 계속 알람이 울린다면 오히려 보행에 방해가 될 수 있다고 생각했습니다.
이런 문제를 해결하기 위한 디바이스로 스마트폰이아닌 로봇을 생각하였습니다.


작동 방식은 로봇강아지에 부착된 카메라가 전후방을 연속적으로 촬영하고 특정 시간 이내에 어떤 객체가 사용자의 방향으로 크기가 점점 커진다면 짓는 소리와 함께 연결된 끈에서 진동을 발생시켜 사용자는 물론 해당 객체 또한 상황을 인지를 할 수 있는 방식입니다.

Azure란 Microsoft에서 만든 클라우드 플랫폼입니다.



Azure 클라우드에는 다양한 서비스가 있습니다.
그중에 저희는 Azure Machine Learning 서비스를 사용하였습니다.



Machine Learning의 기술중 사진을 분석하는 Computer Vision 에는 6가지 학습 모델이 있습니다.

  1. Image Classification은 사진을 분석해서 이미지가 학습된 데이터중 어떤 사진인지 예측하는 모델입니다.
  2. Object Detection은 사진속 객체들을 탐지하여 학습된 데이터들로 분류하는 모델입니다.
  3. Semantic Segmentation은 사진속에서 비슷한 객체들끼리 색으로 표현하는 모델입니다.
  4. Image Analysis는 사진을 분석해서 사진의 상황을 문장으로 설명하는 모델입니다
  5. Face Detection&Recognition은 사진속 사람들의 얼굴을 탐지하는 모델입니다.
  6. OCR(광학문자 인식)은 사진속에서 문자로 된 내용들을 분석하는 모델입니다.


저희는 이 6가지중 Object Detection을 사용하여 이번 프로젝트를 진행하였습니다.

차별점



사실 저희 얘기를 들으시면서 이런 생각이 드실 수 있습니다.
‘이미 개발이 많이 진행 된 뻔한 주제 아닌가?’
맞습니다.
현대의 로봇개나 테슬라의 테슬라 봇처럼 인공지능이 적용된 로봇은 현 시대에 가장 집중하는 분야 중 하나입니다.
하지만 일반적인 시각장애우들이 이런것들로 도움을 받을 수 있을까요?
아닙니다.


왜냐면 금전적 부담이 크기 떄문입니다.
저희는 어떤식으로 제작을 해야 이런 부담을 조금이라도 줄일 수 있을까를 생각했습니다.


그래서 오픈소스와 라즈베리파이를 생각하였습니다.
Azure를 사용해 만든 학습 데이터를 라즈베리파이에 적용시켜 기체를 제작하여 개발 비용을 크게 줄이고 더 많이 보급될 수 있도록 하자는 생각입니다.

제작과정


우선 Azure Custom Vision인 Object Detection을 사용하여 학습을 진행하였습니다.


학습 결과는 위와 같습니다.


학습 결과를 바탕으로 해당 이미지를 테스트 했을 때 이런 결과가 추출되는데 어떤 객체가 있는지, 해당 객체의 위치정보와 크기정보를 알려주게 됩니다.
이 정보를 통해 특정 시간내에 사용자의 방향으로 크기가 커지는 객체를 찾아내는 알고리즘을 적용하여 구현할 생각입니다


하드웨어는 라즈베리파이를 중심으로 제작하였습니다.
라즈베리파이는 네트워크 통신이 가능할 뿐 아니라 5V전압으로 외장베터리로도 구동이 가능해 저희가 생각한 기체에 적용시키기 적합하다고 생각하였습니다.
이런식으로 개발비용을 낮춘다면 보급률을 올릴 수 있고 시각장애인 뿐 아니라 도움이 필요한 다른 분들에게도 조금이나마 의지할 수 있는 버팀목이 될 수 있을 것입니다.
